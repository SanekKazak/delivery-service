## User service

REST API endpoints:

####

    GET /api/users - User profile

####

    DELETE /api/users - Delete profile

####

    POST /api/users- Register user

####

    GET /api/cards/details/{userId} - User cards

####

    GET /api/cards/{cardId} - User card

####

    POST /api/cards/{userId} - Register card

####

    DELETE /api/cards/{userId}/{cardId} - Delete card

####
